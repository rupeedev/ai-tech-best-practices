  Hugging Face MCP allows you to:
  - Search models, datasets, and spaces
  - Access model information and metadata
  - Download model files
  - Explore ML/AI resources
  - Get model documentation

## Common Commands

### Search Models
mcp__hugging-face__search_models
  - query: "text-generation"
  - filter: "task"

### Get Model Info
mcp__hugging-face__get_model
  - model_id: "meta-llama/Llama-2-7b"

### Search Datasets
mcp__hugging-face__search_datasets
  - query: "sentiment analysis"

### List Spaces
mcp__hugging-face__list_spaces
  - sort: "likes"
  - limit: 10
