## Commands Used (Sequential)

### Step 1: Create Project Documentation Page

Tool: `mcp__notion__notion-create-pages`

```json
{
  "pages": [
    {
      "properties": {"title": "Flask App - DigitalOcean Deployment"},
      "content": "# Flask App - DigitalOcean Deployment\n\n## Project Overview\n..."
    }
  ]
}
```

Returns: page_id (e.g., `2cdec5a6-64f7-81e2-a8d5-cd3b9dfb7a83`)

### Step 2: Create Task Database (under the documentation page)

Tool: `mcp__notion__notion-create-database`

```json
{
  "parent": {"type": "page_id", "page_id": "<page_id_from_step_1>"},
  "title": [{"type": "text", "text": {"content": "Flask App Tasks"}}],
  "properties": {
    "Task": {"type": "title", "title": {}},
    "Status": {
      "type": "select",
      "select": {
        "options": [
          {"name": "To Do", "color": "gray"},
          {"name": "In Progress", "color": "blue"},

    },
    "Priority": {
```
