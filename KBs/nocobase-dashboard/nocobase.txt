NocoBase Overview                                                                                                                     
                                                                                                                                        
  NocoBase is an extensible no-code/low-code platform for building business apps with AI capabilities.                                  
                                                                                                                                        
  Key Features:                                                                                                                         
  - Data model-driven (UI decoupled from data)                                                                                          
  - Built-in AI employees (translator, analyst, assistant)                                                                              
  - Plugin-based architecture (like WordPress)                                                                                          
  - Visual page builder                                                                                                                 
                                                                                                                                        
  Quick Setup (Docker)                                                                                                                  
                                                                                                                                        
  Option 1: Standalone                                                                                                                  
                                                                                                                                        
  # Create directory                                                                                                                    
  mkdir nocobase && cd nocobase                                                                                                         
                                                                                                                                        
  # Run with SQLite                                                                                                                     
  docker run -d \                                                                                                                       
    --name nocobase \                                                                                                                   
    -p 13000:80 \                                                                                                                       
    -v ./storage:/app/nocobase/storage \                                                                                                
    nocobase/nocobase:latest                                                                                                            
                                                                                                                                        
  Access at: http://localhost:13000                                                                                                     
                                                                                                                                        
  Option 2: Docker Compose (Recommended)

  NocoBase vs Metabase                                                                                                                  
                                                                                                                                        
  | Feature    | NocoBase       | Metabase      |                                                                                       
  |------------|----------------|---------------|                                                                                       
  | Purpose    | Build apps     | Dashboards/BI |                                                                                       
  | Data entry | Yes            | No            |                                                                                       
  | Forms      | Yes            | No            |                                                                                       
  | Charts     | Yes            | Yes (better)  |                                                                                       
  | Workflows  | Yes            | No            |                                                                                       
  | Use case   | Internal tools | Analytics     |  


  raphiti is for knowledge graph memory (storing relationships/context), not dashboards. Here's a comparison:                          
                                                                                                                                        
  Comparison for AI-Powered Dashboards                                                                                                  
                                                                                                                                        
  | Tool     | AI-Powered      | Dashboard  | Data Entry | Best For                   |                                                 
  |----------|-----------------|------------|------------|----------------------------|                                                 
  | Graphiti | ✅ LLM          | ❌         | ❌         | Memory/context for Claude  |                                                 
  | Metabase | ❌              | ✅         | ❌         | BI analytics               |                                                 
  | NocoBase | ✅ AI employees | ✅         | ✅         | Internal apps + dashboards |                                                 
  | MindsDB  | ✅ ML/AI        | ⚠️ Limited | ❌         | AI predictions on data     |

  My Recommendation                                                                                                                     
                                                                                                                                        
  For AI-powered dashboard: Use NocoBase                                                                                                
                                                                                                                                        
  Why:                                                                                                                                  
  - Built-in AI employees (analyst, assistant)                                                                                          
  - Natural language queries                                                                                                            
  - Visual dashboard builder                                                                                                            
  - Can connect to your existing data                                                                                                   
  - Forms + workflows included                                                                                                          
                                                                                                                                        
  Graphiti is useful for a different purpose - giving Claude memory of your conversations and data relationships. You can use it        
  alongside NocoBase.                                                                                                                   
                                                                                                                                        
  Use Graphiti For                                                                                                                      
                                                                                                                                        
  "Remember our app has 3 endpoints and uses PostgreSQL"                                                                                
  "What did we decide about the dashboard architecture?"                                                                                
                                                                                                                                        
  Use NocoBase For                                                                                                                      
                                                                                                                                        
  - Build admin panels                                                                                                                  
  - Create data entry forms                                                                                                             
  - AI-powered data analysis                                                                                                            
  - Visual dashboards        

                                                                                                                                        
  |              | Graphiti/GraphRAG                              | NocoBase                            |                               
  |--------------|------------------------------------------------|-------------------------------------|                               
  | Purpose      | AI memory & context                            | Build apps & dashboards             |                               
  | What it does | Stores relationships, enables AI to "remember" | Visual app builder with AI features |                               
  | Output       | Knowledge for LLM queries                      | Web apps, forms, dashboards         |                               
  | Use case     | "What did we discuss last week?"               | "Build me an admin panel"           |                               
                                                                                                                                        
  When to Use Each                                                                                                                      
                                                                                                                                        
  Graphiti/GraphRAG                                                                                                                     
                                                                                                                                        
  You: "Remember that our Flask app uses PostgreSQL and has 3 endpoints"                                                                
  Later...                                                                                                                              
  You: "What database does our app use?"                                                                                                
  AI: "PostgreSQL" (retrieved from knowledge graph)                                                                                     
                                                                                                                                        
  Best for: AI memory, context retention, relationship queries                                                                          
                                                                                                                                        
  NocoBase                                                                                                                              
                                                                                                                                        
  You: Build a dashboard showing user signups, revenue, API usage                                                                       
  Output: Visual dashboard with charts, filters, AI analysis                                                                            
                                                                                                                                        
  Best for: Building actual applications and dashboards                                                                                 
                                                                                                                                        
  Use BOTH Together

  │  "What do we       │  Visual charts             │                                                                                   
  │   know about       │  Data entry forms          │

  │                    │                            │

  └─────────────────────────────────────────────────┘                                                                                   
                                                                                                                                        
  Summary                                                                                                                               
                                                                                                                                        
  | Need                         | Use      |                                                                                           
  |------------------------------|----------|                                                                                           
  | AI-powered dashboard UI      | NocoBase |                                                                                           
  | AI memory/context for Claude | Graphiti |                                                                                           
  | BI analytics/charts          | Metabase |                                                                                           
  | ML predictions               | MindsDB  |                                                                                           
                                                                                                                                        
  Bottom line: Use NocoBase for your dashboard. Use Graphiti if you want Claude to remember project context across sessions.
